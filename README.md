# OpenGL-CMAKE

Requires CMake, GLFW >= 3.2, OpenGL >= 3.3, A C++11 compiler

#Linux / OSX
```bash
$ git clone https://github.com/keringar/opengl
$ cd ./opengl/bin
$ cmake ..
$ make
$ ./OpenGL
```
